## ‚öôÔ∏è Setup for the Animated Snake (GitHub Action)

> Create the file **`.github/workflows/snake.yml`** in your `idrees538/idrees538` repo:

```yml
name: Generate Snake

on:
  schedule:
    - cron: "0 0 * * *"   # every day at 00:00 UTC
  workflow_dispatch:      # manual trigger

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Generate Snake SVG
        uses: Platane/snk@v3
        with:
          github_user_name: idrees538
          outputs: |
            dist/snake.svg

      - name: Push Snake to output branch
        uses: crazy-max/ghaction-github-pages@v4
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```
